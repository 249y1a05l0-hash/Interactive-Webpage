# Interactive Web Page (HTML, CSS, JavaScript)

## Description
This project demonstrates a basic interactive web page using HTML, CSS, and JavaScript. It includes simple form validation and JavaScript event handling.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User input form with name and email
- Form validation (fields cannot be empty)
- Interactive message display on submission

## How to Run
1. Open `index.html` in any web browser
2. Fill the form and click submit to see interactive feedback
